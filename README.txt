== Description ==

**An easy-to-use, comprehensive, lightweight, stable and frequently updated freemium image compression plugin supported by the friendly team that created it.**

Increase your website's SEO ranking, number of visitors, and ultimately your sales by optimising any image on your website.
Louis Opticture is an easy to use, lightweight, a plugin that can compress all your past images with a single click. New images are automatically resized/rescaled and optimized on the fly, in the background. It's also compatible with any gallery, slider or eCommerce plugin.

Both lossy and lossless image compression are available for the most common image types (JPG, PNG, GIF, WebP, and AVIF).
We also offer **NearLossless** JPEG compression which is a very high-quality lossy optimization algorithm. Specially designed for photographers!
Optimized images mean better user experience, better PageSpeed Insights or GTmetrix results, better Google PageRank, and more visitors.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `louis-image-compressors` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
